# Task 2. Ansible
Ansible playbook.
### Требования.
- [x] Создание пользователя на удаленной машине
- [x] Авторизация SSH по ключам для пользователя.
- [x] Отключение авторизации по паролю.
- [x] Создание директории в /opt/ с правами для пользователя.
### Заметки.
Сначала сделала создание пользователя. Слева управляющий сервер, справа управляемый. Справа показан вывод ``cat /etc/passwd``:  
![создание юзера](./screenshots/Создание%20юзера.png)  

Дальше сгенерировала ключи командой ``ssh-keygen -t rsa -b 4096``:  
![генерация ключей](./screenshots/Генерация%20ключей.png)  

На удаленной машине посмотрела содержимое файла ``/etc/ssh/sshd_config``. Для отключения авторизации по паролю нужно изменить выделенную строку в ``no``:  
![cat /etc/ssh/sshd_config](./screenshots/Вывод%20sshd_config.png)  

Ну а дальше дополняла плейбук. Создание юзера уже есть, потом сделала передачу открытого ключа, выключила авторизацию по паролю, перезапустила ssh и создала директорию, у которой владельцем является созданный пользователь и права у него 755. Получилось, конечно, не сразу, да и скриншоты в процессе больше не делала, но вот есть скрин финального результата:
